    • bold: <b>

    • subscripted: <sub>

    • Ordered Lists

  5. Tables
    • Consist of a <thead> and a <tbody> block for columns and then row entries, respectively. Enclosed in an OUTER <table> block. Kinda like the head and body of an HTML block! :)
    • Every single column defintion and row entry is enclosed in a <tr> block.
    • Inside each column <tr> block goes 1 element: The column name in a <th> block.
    • Inside each row entry <tr> block go a few elements, matching each column, in <td> blocks each.
    • NOTE: td stands for table data. tr stands for table row.
  6. Links
    • Click on a text or an image to go to a website:

      <p><a href="http://www.google.com"><img src="space_girl.jpg"></a></p>

      NOTE: a stands for Anchor Link.
    • Click on a text or image to jump to another place in the website (eg. top)

      <p id="top"><a href="http://www.google.com"><img src="space_girl.jpg"></a></p>
      <p><a href="#top"> Back to top</a></p>

      NOTE: The paragraph is given an ID. Then the href of a link is given that ID with #. Simple as that.
  7. Entities
    They're just symbols that are given a name and a number in order to be displayed on an HTML page, which would otherwise be HTML code itself.
    For example &lt is <, &gt is > and &amp is &.
