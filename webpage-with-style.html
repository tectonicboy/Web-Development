NOTES:
  • There are 3 ways to write CSS: Inline in HTML, internal in HTML file, or in a separate file (best).
  • You can assign divs, p's, h's and other things CLASSES and IDs. ID you only use once, Classes many times. EVERYTHING IS INSIDE a <style type="text/css"> ... </style> block inside the head of HTML.
  • To define the styling for a class, use
    .class_name {
     styles
    }
  • To define the styling for an ID, use
    #id_name {
     styles
    }
  • Common stylings.
    • font-family: 'Josefin Sans';
    • font-size: 300%;
    • font-weight: bold;
    • background-color: yellow;
    • color: green;
    • color: rgb(179, 230, 255);
    • text-decoration: underline;
    • width: 75%;
    • width: 100px;
    • opacity: 0.5; [NOTE: The smaller the opacity, the more see-through it is.]
    • margin: 50px [Apx, Bpx, Cpx, Dpx, A meaning top side, going clockwise.]
    • padding: 10px [Apx, Bpx, Cpx, Dpx, A meaning top side, going clockwise.] Opposite of Margin - works INSIDE the element.
    • border-[style, width, radius, color]
  • You can include PART of a paragraph to be in a class:

    <p>bla bla bla... <span class="class_name">Text is part of the class.</span> </p>

  • A thing can be part of multiple classes, not just one.
  • Positionings.
    1. Floats.
      • We float something to the left by including a float: left; style. Similarly with Right Float.
      • To make something appear below everything that's floated, we say clear: both; in its styles.
    2. Relative Positioning.
      • To make something movable relative to where it would normally have been we say position: relative; left: 75px; top: -135px; in their styles section.
      • NOTE that negative values mean positive movements!
      • We let elements have a z-index in their styles section in order to determine which one appears on top of which. Higher value means more on top.
    3. Absolute Positioning.
      • Removes an element from the flow of the website. Moves relative to the top-left of the page.
    4. Fixed Positioning.
      • Still out of the flow of the websites, but stays in position as we scroll. Useful for a menu or something like that.
    5. For all 3 Positioning types, we use left and top to move it wherever we want.
  • Special Classes.
    • You can style the whole Body of the webpage by the syntax:
      body {
       styles
      }
    • For example, to get rid of the annoying white spaces in the webpage (default margin and padding values), we set both to 0px in body styling block.
    • Similarly, we can set them both to 0px for paragraphs, lists, etc, so there's no space between them either.
  • WARNING: Different browsers may have different default margin and padding values. To make sure
    your site looks right on all browsers, use something called CSS Resets.
  • BORDERS are extremely important when making an aesthetically pleasing box or circle with things in it! colors, width, radius, styles.
